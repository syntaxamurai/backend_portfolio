## About

Personal portfolio site built with plain HTML, CSS, and JavaScript — no frameworks, no build step.
Designed with a dark terminal aesthetic and fully responsive across all screen sizes.

## Features

- Light / dark theme toggle — preference saved in localStorage
- Fully responsive — mobile burger menu, desktop nav
- Terminal SVG illustration — pure inline SVG, no images
- Active nav link highlighting per page
- CV download button
- Certificate links on education page

## Local Setup

No installs needed.
```bash
git clone https://github.com/syntaxamurai/backend_portfolio.git
cd portfolio
```

Open `index.html` in your browser — or use the Live Server extension in VS Code for auto-reload on save.

---

## Deployment

Hosted on GitHub Pages.
```
Settings → Pages → Deploy from branch → main / root → Save
```

Live at: `https://syntaxamurai.github.io/backend_portfolio/`
